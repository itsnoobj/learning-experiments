# Logo: The Pattern Grid

## Concept

A 3×3 grid of squares with varying opacity and color (charcoal + gold). Some cells filled, some faint, some empty — like a behavioral pattern emerging from noise.

## Why It Works

- **Patterns are the product.** The site teaches you to recognize recurring patterns in human behavior. The logo IS a pattern.
- **Abstract, not literal.** No human figures, no bathroom signs, no clichés.
- **Scalable.** 9 squares work at 16×16 favicon. Works at billboard size.
- **No cultural baggage.** A grid is universal.
- **Memorable.** Simple geometric shape — redrawable from memory.
- **Flexible.** The pattern can shift per section (different cells lit up = different "dynamic" revealed).
- **Timeless.** Geometric marks don't date. Think: Microsoft windows, Uniqlo, BBC grid.

## The Narrative

> You're looking at a grid. Some squares are visible, some hidden. That's how human dynamics work — some forces are obvious, most are invisible until you learn to see the pattern.

## Specifications

- 3×3 grid (9 cells)
- Rounded corners on each cell (rx: 4)
- Colors: charcoal `#2C2C2C` + gold `#B8860B`
- Varying opacity across cells (0.2 → 1.0) — pattern "emerging"
- Cream background `#FAF7F2` (or transparent)
- Gap between cells: consistent, clean

## Usage

| Context         | Treatment                                            |
| --------------- | ---------------------------------------------------- |
| Favicon         | Grid only, 16×16, no text                            |
| Website header  | Grid + "humandynamics" wordmark                      |
| Social avatar   | Grid on cream square                                 |
| Loading state   | Cells animate in sequence (pattern revealing itself) |
| Chapter markers | Different grid patterns per Part (I–X)               |

## Variations to Explore

- Static grid with fixed pattern
- Animated grid (cells fade in one by one — "the pattern becomes clear")
- Each Part of the book gets its own grid signature (like a fingerprint per topic)
- Dark mode: gold cells on dark background, charcoal cells become light

## Decision

✅ **Chosen direction.** Abstract, scalable, meaningful, no cliché risk.
